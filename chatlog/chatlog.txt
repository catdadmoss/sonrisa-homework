## **Why You Need an App Password (Not Your Regular Gmail Password)**

### **The Problem with Regular Passwords:**

Google (and other email providers) **block** applications from logging in with your regular password for security reasons. This is because:

1. **2-Factor Authentication (2FA)** - If you have 2FA enabled, apps can't enter the verification code
2. **Less Secure Apps** - Google considers SMTP access "less secure" than OAuth
3. **Account Security** - Your main password shouldn't be stored in config files

### **What is an App Password?**

An **App Password** is a **16-character password** that Google generates specifically for one application. It:

- Works **only for that specific app** (like this Alert System)
- Can be **revoked anytime** without changing your main password
- **Bypasses 2FA** for that specific app
- Is more secure because you're not exposing your main password
